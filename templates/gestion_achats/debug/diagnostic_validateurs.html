Comment résoudre les problèmes
Si un employé n'a pas de compte utilisateur lié :
  1. Allez dans l'interface d'administration Django (/admin)
  2. Recherchez l'employé par son matricule
  3. Dans le champ "User", créez un nouveau compte utilisateur ou liez-en un existant
  4. Assurez-vous que le compte utilisateur est actif
Si le compte utilisateur est inactif :
  1. Allez dans l'interface d'administration Django
  2. Recherchez l'utilisateur
  3. Cochez la case "Actif"
  4. Enregistrez les modifications
